# Profit/Loss Dashboard

Vite + React app for month/annual budget with four pies, drilldowns, edit/delete, exports, and settings.

## Dev
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deploy (Vercel)
- Import this repo in Vercel
- Framework: Vite (React)
- Build Command: `npm run build`
- Output Dir: `dist`
